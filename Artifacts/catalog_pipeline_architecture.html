Output Fork — Three Simultaneous Destinations
🤖
ACP Feed → ChatGPT
OpenAI Agentic Commerce Protocol
Format: JSONL (gzip compressed)
Delivery: HTTPS push to OpenAI endpoint
Cadence: Full feed daily + incremental via API
Freshness: Up to 15-min updates for price/inventory
Required fields: id, title, description, price, availability, image_link, brand, google_product_category
B2B extras: min_quantity, decoration_method, production_time, use_case_tags, group_id
Indexing time: 1–7 days after first submission
Validation: OpenAI spec v2026-01-30
🔍
UCP Manifest → Google
Universal Commerce Protocol
Format: JSON manifest file
Delivery: Published at /.well-known/ucp on distributor domain
Cadence: Google crawls on its schedule (daily+)
Freshness: Manifest updated on price/inventory events
Required fields: Same base as GMC + checkout endpoints
Powers: Google AI Mode + Gemini shopping + Google Search AI Overviews
Foundation: Built on existing GMC feed data
Indexing time: 3–14 days after manifest published
🛒
GMC Feed → Google Shopping
Google Merchant Center
Format: CSV, XML, or Content API
Delivery: Push to Merchant Center account
Cadence: Full feed daily — supplemental feeds for price/inventory
Freshness: Price + availability via supplemental feed (hourly)
Required fields: id, title, description, link, image_link, price, availability, brand, GTIN, google_product_category
B2B extras: unit_pricing_measure, multipack, min_handling_time, custom_labels 0–4
Powers: Shopping ads + free listings + UCP foundation
Indexing time: 24–72 hours
Phase 3 — Continuous Management
✅
This is your subscription revenue. One-time enrichment is a project. Continuous management is a platform. The distributor cannot turn this off without losing their AI visibility — that's the stickiness that justifies $1,500–3,000/month recurring.
⚡
Real-Time Triggers
EVENT-DRIVEN — minutes
  • Price change detected → push incremental ACP update immediately
  • Inventory goes to zero → update availability flag in all three feeds
  • New SKU onboarded → trigger enrichment pipeline automatically
  • SKU discontinued → remove from feeds + flag to customer
  • PromoStandards inventory webhook → real-time sync for in-stock status
  • ACP feed rejection alert → auto-diagnose reason and attempt fix
📅
Daily Jobs
SCHEDULED — nightly
  • Full ACP feed regeneration and push (replaces previous full file)
  • GMC full feed refresh and supplemental price/inventory overlay
  • UCP manifest regeneration with latest pricing
  • Feed validation report — flagging any new disapprovals or warnings
  • Price accuracy check — feed price vs. live website scrape comparison
  • New PromoStandards product data pull — detect supplier catalog updates
📊
Weekly Jobs
SCHEDULED — Sunday nights
  • Citation monitoring — run 50 buyer-intent queries across ChatGPT + Gemini
  • Share of voice report — which SKUs appear, at what position, vs. competitors
  • Zero-result query detection — buyer queries returning no distributor results
  • Enrichment gap report — which SKUs are still uncited and why
  • GMC performance pull — impressions, clicks, CTR by SKU from Merchant Center
  • New SKU enrichment batch — any SKUs added that week processed automatically
📈
Monthly Jobs
SCHEDULED — 1st of month
  • Full re-enrichment of bottom 20% scoring SKUs — refresh descriptions
  • AI-referral revenue attribution report — sessions, conversions, revenue by source
  • Competitor visibility benchmark — how client citation rate compares to competitors
  • ACP spec version check — OpenAI updates spec quarterly, ensure compliance
  • PromoStandards catalog delta — new suppliers onboarded, new endpoints available
  • ROI report delivery — the document that justifies renewal
